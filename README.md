# Customer Sentiment LSTM App

An end-to-end Sentiment & Emotion Analyzer:
- Bidirectional LSTM (3-class: Negative / Neutral / Positive)
- Streamlit UI: Single, Bulk, Audio input
- Wordclouds, keyword highlighting, confidence scores
